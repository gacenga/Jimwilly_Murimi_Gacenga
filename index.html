Sentiment Pro

Status:Ongoing

This project involves collecting tweets related to a specific topic and performing sentiment analysis on them. It aims to understand the overall sentiment (positive, negative, neutral) associated with the topic on Twitter. By utilizing Python, Tweepy for accessing the Twitter API, and TextBlob for sentiment analysis, it seeks to instill an understanding of public sentiment and opinions. Through analysis and insights, the project fosters data-driven decision-making and promotes the application of sentiment analysis in various fields.
